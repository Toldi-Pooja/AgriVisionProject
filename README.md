# AgriVisionProject 🌱

## Overview

AgriVision is an AI-powered Smart Agriculture System developed using Django, YOLO, TensorFlow, and Deep Learning. The platform helps farmers and agricultural professionals detect crop diseases, identify weeds, and assess crop quality through image-based analysis.

## Features

* 🌾 Crop Disease Detection
* 🌿 Weed Identification using YOLO
* 📊 Crop Quality Assessment
* 👤 User Authentication and Dashboard
* 🛠 Admin Dashboard
* 📁 Dataset Management
* 🤖 Deep Learning Model Integration

## Technologies Used

* Python
* Django
* TensorFlow / Keras
* YOLO
* HTML
* CSS
* SQLite

## Project Structure

* AdminApp – Admin functionalities
* UserApp – User functionalities
* models – Trained AI models
* static – CSS, images, and assets
* templates – HTML templates

## Installation

```bash
git clone https://github.com/Toldi-Pooja/AgriVisionProject.git
cd AgriVisionProject

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
